# hjacquelin-os
This is an experimentation with Fedora Atomic Desktops, from a Silverblue image.

- install-toolbox is a script that download JetBrains toolbox in order to install IDEs
- comes with Distrobox
- eza for nicer LS display
- zoxide for smarter CD commands
- btop
- jq
- gnome-tweaks
- fastfetch
